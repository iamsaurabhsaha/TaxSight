# AI Tax Prep Assistant

Privacy-first AI tax preparation tool with BYOK (Bring Your Own Key) LLM support.

## Features

- Guided tax interview — conversational step-by-step walkthrough
- Document ingestion — upload W-2s, 1099s with OCR + LLM vision
- Deduction & credit finder
- Tax estimation — US federal + all 50 states
- What-If scenario modeling
- PDF summary export
- Supports Claude, OpenAI, Gemini, or Ollama (local/private)
- All data stored locally in SQLite
